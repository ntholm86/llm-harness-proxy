  THE CEILING -- NAMED HONESTLY
  ------------------------------

                  +---------------------------------+
                  |                                 |
                  |   INTERNAL COMPUTATION          |
                  |   (weights, activations,        |
                  |    causal circuits)             |
                  |                                 |
                  |   ?  unobservable at scale  ?   |
                  |                                 |
                  +----------------+----------------+
                                   |
                                   |  generates
                                   v
                  +---------------------------------+
                  |                                 |
                  |   THINKING TOKENS               |
                  |   (extended reasoning blocks)   |
                  |                                 |
                  |   <-- captured by proxy         |
                  |       BEST AVAILABLE SIGNAL     |
                  |                                 |
                  +----------------+----------------+
                                   |
                                   |  produces
                                   v
                  +---------------------------------+
                  |                                 |
                  |   FINAL RESPONSE                |
                  |                                 |
                  |   <-- captured by proxy         |
                  |   <-- captured by AILANG        |
                  |   <-- signed by Sharif's chain  |
                  |                                 |
                  +----------------+----------------+
                                   |
                                   |  agent then writes
                                   v
                  +---------------------------------+
                  |                                 |
                  |   SELF-REPORTED REASONING       |
                  |   (the trail entry the agent    |
                  |    composes after the fact)     |
                  |                                 |
                  |   X  most vulnerable to         |
                  |      post-hoc rationalization   |
                  |                                 |
                  +---------------------------------+


  Thinking tokens are the closest signal to internal computation
  currently observable. They are not the computation itself.
  Whether they faithfully reflect the weights that produced them
  is an open research question -- Anthropic's interpretability work
  is the frontier and cannot yet produce a causal account of a
  specific frontier-model inference at production scale.

  This ceiling is shared by every approach in the stack.
  It is the correct scope boundary, not a design failure.
